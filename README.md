# 강의소개 
스프링부트를 활용하여 게시판을 만드는 강의입니다. 
초보분들에게 적합한 내용으로 구성하였습니다. 게시판의 최소기능인 글작성, 글목록, 글조회, 글수정, 글삭제 기능을 구현하였습니다.
